■ IT
IT
Infomation Technology の略。情報技術(パソコンやスマートフォンなど)の総称

ICT
Infomation and Communication Technology の略。ITとほぼ一緒の意味で情報共有のコミュケーションの意味合いが強い

IoT
Internet of Things の略。身の回りのすべてのモノがインターネットにつながる仕組み

■ OS
OS
Opreating Systemの略。システム全体を管理し、アプリケーションを動かすための最も基本的なソフト

Windows
Microsoft社のつくったosでいろんなメーカが作成したパソコンに乗せている。幅広い

macOS (Mac OS X)
Apple社の作ったOSでハードウェアからソフトウェアまでApple社で面倒を見ている。使いやすい

UNIX
macOSのもとのOS。最古にちかいOSで、UNIXをもとにしたOSは多々ある。そういう意味で現役

Linux
オープンソース(無料)で使え、中身もいじれるため企業サーバーとして使われることが多い。androidなども
■ 言語・環境
マークアップ／フロントエンド
HTML、CSS、JavaScriptなどを使用しながら、テキストや装飾をコンピューターが正しく認識できるように意味づけを行っていくこと。

HTML
ハイパーテキスト・マークアップ・ランゲージ（Hyper Text Markup Language）の略。表示したい文章や写真などの情報をコンピューターに文章構成を支持すること

CSS
カスケーディング・スタイル・シート（Cascading Style Sheets）の略。webページのスタイルを指示する言語

JavaScript
webページを構成するすべてのオブジェクトを操作できる。例えばメールアドレスをフォームに入力したかどうかチェックする、など　

jQuery
JavaScriptでできることをより簡単な記法で実現できるようにせっけいされたJavaScriptライブラリ

Java SE
Java Platform, Standard Edition の略。Javaで使用されるAPIをまとめたもの
サーバーサイド　インターネットのサービスをクライアント(PCやスマホ)側とサーバ(サービスを提供)側と考えたときのサーバ側のこと

.NET（C#, VB）
主にMicrosoft .NET Framework（マイクロソフト ドットネット フレームワーク）というアプリケーション、または開発環境を指す単語

Java
プログラミング言語のひとつ。どんな環境でも動く

Java SE
Javaを使うための最も基本的なAPI。表示するための関数

Java EE（Jakarta EE）
Javaを使うためのちょっと高級なAPI。web上で動く関数

Perl
プログラミング言語のひとつ。後方互換（アップデートに強い）が強い

Ruby
日本で開発されたオブジェクト指向のスクリプト言語。webサイト、SNSの開発などで使われている

nginx

■ SQL
SQL
データベース（RDBMS）を操作するための言語

DB2
IBM社が提供しているデータベース管理システム（RDBMS）の総称

MySQL
Oracleが開発してサポートするオープンソースのデータベース管理システム（RDBMS）

Maria DB
MySQL派生として開発されている、オープンソースの関係データベース管理システム (RDBMS) 

Oracle DB　
Oracleが開発・販売しているデータベース管理システム（RDBMS）

PostgreSQL　
拡張性とSQL準拠を強調するフリーでオープンソースの関係データベース管理システム（RDBMS）

SQLite　
パブリックドメインの軽量な関係データベース管理システム (RDBMS)。アプリケーションに組み込んで使用される

SQL Server　
Microsoftが開発している関係データベース管理システム (RDBMS)。

CRUD　
ほとんどのコンピューターが持つ基本性能のイニシャルを並べたもの。、Create（生成）、Read（読み取り）、Update（更新）、Delete（削除）

トランザクション　
複数の作業を、一つの作業としてまとめるもの

コミット　
トランザクション処理において、関連する処理がすべて成功し、結果を確定させること　

ロールバック　
障害発生時に処理途上でコミットされていないトランザクションの処理をすべて取り消し、最初の時点に戻すことを意味する。

排他制御（排他ロック）
ファイルへの書き込みが行われている間、別のプログラムによる書き込みを禁止する制御

楽観ロック　
同時更新は起きないだろうという楽観的な前提の排他制御。データそのものにロックはかけない

悲観ロック　
悲観的な前提の排他制御。更新対象のデータを取得する際にロックをかける

実行計画　
SQLを実行するための計画

チューニング　
設定や攻勢を調整し、目標の状態に近づけたり性能を採泥減引き出す作業

パフォーマンス　
コンピュータなどの処理能力や実行速度など

スロークエリ　
データベース管理システム（DBMS）に対する問い合わせ（クリエ）のうち遅いもの

クエリログ　
MySQL サーバが実行した全ての SQL クエリを出力するログです。

IDE（統合開発環境）　
ソフトウェア開発に必要なソフトウェアを一つに組み合わせ、同じ操作画面から統一的な操作法で利用できるようにしたもの

Eclipse
エクリプス　オープンソースの統合開発環境

NetBeans　
ネットビーンズ　オラクルを中心としたコミュニティで開発されたオープンソースの統合開発環境

SpringToolSuite　
Java開発環境のデファクトスタンダードであるEclipseをベースに、Spirng Framework利用のための各種プラグインを追加して作られたSpringオリジナルな開発ツール

DBツール　
データベース管理タスクに使用できるツール、補助ツールの総称

A5:SQL Mk-2　
SQL文の入力支援やER図作成などの機能を備えた高機能なSQL開発環境

pgAdmin　
GUI の管理インタフェースである。 PostgreSQL License で配布される オープンソースソフトウェア

DBeaver
Oracle SQL、SQL Server、PostgreSQL、MySQL、MariaDB、SQLiteなどほぼ全てのDBに対応したSQLクライアントツール。

MySQL Workbench
MySQLの公式サイトにてMySQL Serverと共に配布されている、データベースの設計・開発・管理を一本で実現している無料ツール。

Sequel Pro
データベースをExcelのような表で確認・編集をすることができるMac専用のアプリ

TeamSQL
SQLクライアントツール。サービスは終了してる？

■ UI（ユーザーインターフェース）
UI　
機器やソフトウェア、システムなどとその利用者の間で情報をやり取りする仕組み。

CUI
Character-based User Interface（キャラクタユーザインターフェース）の略。UIの類型の一つで、すべてのやり取りを文字によって行う方式

GUI
Graphical User Interface（グラフィカルユーザーインターフェース）の略。
UIの分類の一つで、情報の提示に画像や図形を多用し、基礎的な操作をマウスやタッチスクリーンなどによる画面上の位置の指示により行うことができるもの。

■ ファイル／ディレクトリ
ファイル　
コンピュータのデータ管理単位の一つ

拡張子
ファイル名の一つ。『.』で区切られた一つ

ディレクトリ
多数の対象をその所在などの情報と共に一覧できるよう整理したもの

ルート・ディレクトリ
コンピュータがストレージ内容を整理するファイルシステムにおいて、最上位のディレクトリ

カレント・ディレクトリ
実行中のソフトウェアなどがストレージのファイルシステム中で現在地として示すディレクトリ

パス
コンピュータ内でファイルなどの所在地を示す文字列

絶対パス
階層構造の頂点（最上位階層）からの位置関係を記述する方式。

相対パス
現在位置からの相対的な位置関係を記述する方式。

LAN

WAN

クライアント

コンピュータ、ソフトウェア

サーバー

■ 各種プロトコル
HTTP
WebサーバとWebクライアントの間でデータの送受信を行うために用いられるプロトコル（通信規約）

FTP
インターネットなどのTCP/IPネットワークでファイル転送を行うことができるプロトコル（通信規約）

HTTPS
通信方式の種別などを表すURIスキームの一つ、WebサーバとWebブラウザの間の通信が暗号化されていることを意味し、通信経路上での盗聴や改竄、第三者によるなりすましを防止する。

SSL
インターネットなどのIPネットワークでデータを暗号化して送受信するプロトコル（通信手順）

SSH
主にUNIXコンピュータで利用される、ネットワークを介して別のコンピュータにログインして操作するためのソフトウェアの一つ。

ポート、ポート番号
コンピュータ上で実際にデータ通信を行うための出入口の番号

HTTP、FTP、SSHの各ポート番号
TCP 20 : FTP (データ)
TCP 21 : FTP (制御)
TCP 22 : SSH
TCP 80 : HTTP

ファイアウォール
外部の攻撃から内部を保護するためのソフトウェアや機器、システムなどのこと

TCP
インターネットなどのネットワークで、IP（Internet Protocol）の一段階上位層のプロトコル（通信規約）

■ IPアドレス
IPアドレス
インターネットなどのTCP/IPネットワークに接続されたコンピュータや通信機器の一台ごとに割り当てられた識別番号

グローバルIPアドレス
インターネットに直に接続された機器に割り当てられるIPアドレス

プライベートIPアドレス
企業など組織の内部で運用されるネットワーク上で各機器に割り当てられるIPアドレス。

■ ドメイン
ドメイン
インターネットなどのTCP/IPネットワーク上で機器やネットワークを識別するドメイン名（domain name）を指すことが多い

DNS
インターネットなどのIPネットワーク上でドメイン名（ホスト名）とIPアドレスの対応関係を管理するシステム

VPN
通信事業者の公衆回線を経由して構築された仮想的な組織内ネットワーク

ブリッジ
受信したデータのMACアドレスなどデータリンク層（リンク層、第2層、MAC層）の宛先情報を参照して中継の可否を判断する機能

プロキシサーバー
内部のコンピュータの「代理」（proxy）としてインターネット上のコンピュータへ接続を行う

■ 電子メール
SMTP
インターネットなどのTCP/IPネットワークで標準的に用いられる、電子メール（eメール）を伝送するための通信手順（プロトコル）

POP
メールサーバのメールスプールに動的にアクセスするために用いられるプロトコル

IMAP
インターネットなどのIPネットワークで標準的に用いられる、電子メール（eメール）を受信するための通信規約（プロトコル）

■ セッションとクッキー
セッション
接続/ログインしてから、切断/ログオフするまでの、一連の操作や通信のこと

クッキー
Webサイトの提供者が、Webブラウザを通じて訪問者のコンピュータに一時的にデータを書き込んで保存させる仕組み。

キャッシュ
直近に読み込んだものや使用頻度が高いものを高速な記憶装置に複製しておくこと、複製されたデータそのもの

■ バージョン管理
Git
オープンソースの分散バージョン管理システム
複数の開発者が共同で一つのソフトウェアを開発する際などに、ソースコードやドキュメントなどの編集履歴を統一的に管理する

SVN
Subversion　オープンソースのバージョン管理システムの一つ。複数の開発者が共同で一つのソフトウェアを開発する際などに、ソースコードやドキュメントを管理する

デグレ

マージツール
